 # Le Matin
Morning or Le Matin is a personal website focused on showing how to make mornings the best way to start your day, with advices, fun-facts and also pretty cool breakfast ideas that you can enjoy all the times you like.

You can try out the website through this link
https://isabellagg19.github.io/Morning/

To make this website I used
html, to give it structure and order
css, to make it look unique and amazing
js, to give it that special touch and make it more fun

Explore this website every time you need that extra motivation during the first part of the day.

In the making I learned a lot of incredible things I didn't know about mornings, which was amazing, and made me a even bigger fan of mornings, and wen through coding challenges such as making the buttons work, which was a little bit hard, and make it look the way I wanted to.
After all the work I really liked the result.
Hope you enjoy this website as much as I enjoyed making it.
